# DEVLOG — EMT-1 練習工具

## 2026-06-28 — 新增 `drill.html`：實境演練器（情境一步一步）

### 背景
原本只有 `index.html`（73 題情境選擇題庫）。隔天有 EMT-1 **最終實作**考試：考官出實境情況 → 一步一步做到輔助檢查 → 與到場救護同仁交接才算完成。需要一個能投影到大螢幕、考前一起演練的工具。

### 產出：`drill.html`（單檔、可離線、放同一個 GitHub Pages）
- 線上網址（push 後）：`https://inorino.github.io/emt1-quiz/drill.html`
- 不需新 QR code，原網址後接 `/drill.html` 即可。

### 設計重點
1. **路人視角開場**：情境不點破創傷／非創傷與病因，學員要自己從現場判斷機轉。非創傷個案都安排家屬／旁人在場，問病史(SAMPLE)時找得到人。
2. **二段式考官回饋**（避免爆雷）：
   - 標準動作預設「隱藏」（同學口述）；需要對照才用右上開關「顯示」。
   - 點①顯示「考官回饋（現場狀況）」→ 學員處置 → 點②顯示「考官回饋（處置後結果）」→ 重點延到最後才出現。
3. **兩套流程**（依教官標準）：
   - 創傷：現場 → 意識 → A·B·C·D·E → 病史 → 輔助檢查 → 危急度 →（有頸椎機轉才上頸圈）→ 交接
   - 非創傷：現場 → 意識 → A·B·C → 輔助檢查 → 病史 → 危急度 → 交接（無 D/E/頸圈）
4. **頸圈判斷**：只有「創傷且有頸椎受傷機轉」才上頸圈；創傷但無機轉（如切割傷）／非創傷皆不上 → 含對照案例。
5. **跑完全局複習**：左欄整條流程、右欄點開看該步詳解（標準動作＋考官回饋①②＋重點），手機自動上下堆疊。
6. 字級縮放鈕（100/125/150/200%，記憶設定）、🎲 隨機抽題、創傷🔴/非創傷🔵 標籤。

### 20 個情境
- 創傷(7)：機車事故、高處墜落、廚房切割傷、樓梯撞頭、汽車對撞、腹部穿刺、熱液燙傷
- 非創傷(13)：OHCA(CPR)、低血糖、中風、溺水、胸痛、氣喘、抽搐、蛇咬、蜂螫過敏、中暑、CO中毒、腸胃道出血、酒醉藥物

### 內容來源
所有步驟與數值對照 `index.html` 題庫（使用者已驗證的教官標準），未自行加入未經確認的 EMT 流程。臨場仍以教官與現行救護作業規範為準。
